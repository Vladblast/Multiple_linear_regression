# Multiple_linear_regression
Анализ датасета с помощью множественной линейной регрессии.

Кратко
- Репозиторий содержит Jupyter Notebook с пошаговым анализом данных и построением модели множественной линейной регрессии.
- Цель: показать процесс предобработки, EDA, отбор признаков, обучение, оценку качества и интерпретацию результатов.

Содержание
- Основной ноутбук: 
- `data/` — исходные данные,
- README.md

Функциональные шаги в ноутбуке
1. Загрузка и первичный обзор данных: `head()`, `info()`, описательная статистика.
2. EDA: распределения, гистограммы, парные графики.
3. Разделение на трейн/тест.
4. Обучение модели множественной линейной регрессии.
5. Оценка качества: MSE, RMSE, MAE, R²; сравнение истинных и предсказанных значений.
6. Проверка предположений линейной регрессии:
   - линейность,
   - независимость,
   - точночти,
   - значимости.
7. Интерпретация коэффициентов и выводы.

Требования
- Python 3.8+
- Jupyter Notebook/JupyterLab
- Библиотеки:
  - numpy
  - pandas
  - matplotlib
  - seaborn
  - scikit-learn
  - statsmodels (для детальной статистики, опционально)

Установка окружения
```bash
python -m venv .venv
# Linux / macOS
source .venv/bin/activate
# Windows
# .venv\Scripts\activate

pip install --upgrade pip
pip install jupyterlab numpy pandas matplotlib seaborn scikit-learn statsmodels
```

Запуск
```bash
jupyter lab
# или
jupyter notebook
```
Откройте основной ноутбук (`*.ipynb`) и выполните ячейки по порядку.

Использование своего датасета
- Поместите файл в папку `data/` (или укажите корректный путь в ноутбуке).
- Убедитесь, что столбцы/типы данных совпадают с ожиданиями ноутбука.
- При необходимости обновите список признаков и целевую переменную.

Результаты
- Метрики качества (RMSE/R² и др.), графики остатков, важности/веса признаков и интерпретация коэффициентов представлены в ноутбуке.
- Анализ полученной модели ,проверка гипотез о значимости модели и ее коэффициентах.
